# Movie Genre Classification

## Objective
The objective of this project is to build a machine learning model that predicts the genre of a movie based on its plot description.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- TF-IDF Vectorizer
- Multinomial Naive Bayes

## Dataset
IMDb Movie Genre Classification Dataset

## Steps Performed
1. Loaded the training and testing datasets.
2. Preprocessed the movie descriptions.
3. Converted text into numerical features using TF-IDF.
4. Trained the model using the Multinomial Naive Bayes algorithm.
5. Tested the model on the test dataset.
6. Evaluated the model using accuracy.
7. Added user input prediction functionality.

## Result
The model achieved an accuracy of approximately 47%.
