# Spam SMS Detection

## Objective
The objective of this project is to build a machine learning model that can classify SMS messages as Spam or Ham (Normal).

## Technologies Used
- Python
- Pandas
- Scikit-learn
- TF-IDF Vectorizer
- Naive Bayes Classifier

## Dataset
SMS Spam Collection Dataset

## Steps Performed
1. Loaded the dataset.
2. Preprocessed the data.
3. Converted text into numerical features using TF-IDF.
4. Split the dataset into training and testing sets.
5. Trained a Naive Bayes classifier.
6. Evaluated the model using accuracy.
7. Added user input prediction functionality.

## Result
The model achieved an accuracy of approximately 96.54%.
