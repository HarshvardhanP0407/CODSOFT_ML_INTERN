# 📉 Customer Churn Prediction using Machine Learning

## 📌 Project Overview
This project predicts whether a customer will **leave (churn)** or stay based on their behavior and account details.

## ⚙️ Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib / Seaborn (optional)

## 📊 Dataset
Customer data including demographics, account information, and usage patterns.

## 🧠 Model Used
- Logistic Regression / Random Forest

## 🔄 Workflow
1. Data preprocessing
2. Handling missing values
3. Feature encoding
4. Model training
5. Prediction

## 🎯 Result
The model helps identify customers likely to churn.
